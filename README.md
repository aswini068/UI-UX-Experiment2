# UI-UX-Experiment2

## Aim:
To Design a Basic Mobile and Web Layout for an Existing Case Study

## Algorithm:
Apply Gestalt principles to enhance mobile and web layout design. Incorporate UI elements and design patterns into the layouts. Develop interaction behaviors and usability principles for the designs. Create a brand logo using Canva, following styling and branding guidelines and include it with your layouts.

## Output:
Design a Basic Mobile and Web Layout for an Existing Case Study

a. Apply Gestalt principles to enhance mobile and web layout design. For "Dream Crest," Gestalt principles elevate the luxury feel and coherence:
Proximity: Groups related content, e.g., event info and RSVP together, to reduce visual clutter and emphasizerelationships. Similarity: Uses uniform gold accents, serif/scripty typography, and rounded motifs in icons/buttons so the interfacefeels harmonious with the logo’s gold glitter and laurels. Figure-Ground: Employs a black background with rich gold elements to create strong contrast, making keycontent, calls-to-action, and imagery stand out—mirroring the radiance and exclusivity of the logo. Continuity & Closure: Aligns navigation and section dividers in sweeping, continuous lines and uses cardshapes, echoing the laurel wreath, to support visual flow and luxury cues for web and mobile. By combining these, theexperience feels premium, intuitive, and visually captivating.

b. Incorporate UI elements and design patterns into the layouts. Mobile: Use a minimalist bottom navigation bar with gold accent icons, large touch-friendly buttons, andcardlayoutsfor content sections. Floating or sticky CTAs employ gold gradients with elegant shadowing, consistent withthelogo’sopulent art style. Web: Feature a top navigation/header in black with the logo at left, primary links in gold/cream, secondarynavigationina sidebar or dropdown. Employ luxury design patterns: hero banners with soft gold gradients, parallax effects, andfeatured cards with high-res imagery edged in gold. Patterns: Offer modal dialogs or overlays with translucent black-and-gold tint, clean data pickers, and progressiveformsfor exclusive membership/event invites. Pattern selection prioritizes elegance, visual drama, and usability, fit for a premium audience. 

c. Develop interaction behaviors and usability principles for the designs. Interaction: Provide tactile, luxurious feedback—subtle gold shimmer glow or ripple effects on button taps, andsmoothtransition effects (such as fades or slides) for gallery and content reveals. Usability: Use large touch zones on mobile, ensure high color contrast (gold on black), and keep navigationstickyoreasily accessible. Typography remains readable (light-on-dark, large size) to match the brand’s prestige andaccessibility.Personalization: For "Dream Crest," offer profile personalization, wishlists, and possibly exclusive content for logged-inusers via luxurious micro-interactions (e.g., animated user greetings). Consistency: Every element, from iconography to feedback animations, follows the same gold-on-black motif, providinga seamless brand experience across screens.

d. Create a brand logo using Canva, following styling and branding guidelines and include it with your layouts. Designed in Canva, it uses a black background for drama and exclusivity, with gold glitter effects for the figureandtext.The scripty, elegant typeface and circular laurel crown reinforce notions of luxury, celebration, and exclusivity, echoingupper-tier branding conventions. The logo is vector-friendly, scalable for both web headers and mobile app bars, and maintains strong figure-groundcontrast for visual clarity.
